# FUTURE_UX_03 – CRM / Client Management Dashboard UI

## 📌 Project Overview
This project focuses on designing a CRM / Client Management Dashboard UI for a small agency or B2B service-based business. The goal was to create a structured, workflow-based dashboard that improves lead tracking and client management.

## 🎯 Objective
- Design an efficient lead management system
- Improve workflow visibility
- Create a user-friendly dashboard interface
- Structure data for easy decision-making

## 🛠 Design Tool Used
- Figma (Web-based Dashboard UI Design)

## 📊 Screens Designed
1. Dashboard Overview
2. Lead Pipeline (Status-based stages)
3. Client List Management
4. Client Profile View
5. Task & Follow-up Tracker

## ✨ Key Features
- Visual lead pipeline (New, In Progress, Converted, Lost)
- Search & filter options
- Client engagement status tracking
- Task reminders & follow-up scheduling
- Clean data visualization layout

## 🧠 UX Strategy
- Clear visual hierarchy for data readability
- Card-based layout for pipeline stages
- Minimal clutter for better focus
- Logical workflow structure based on real agency operations
